The Smart Walker is equipped with an ultrasonic sensor that detects nearby objects (up to 2 feet) and warns the person by vibrating the upper part of the stick.

It is connected with the user’s mobile phone and updates the number of steps the user has walked.

The stick is also equipped with a GPS Module which can provide real-time location update to the family members of the user.

The Smart Walker+ is also equipped with a heart rate sensor to check the heartbeat of the user which can be tracked via the companion app.

The Smart Walker can also detect a collision. In such cases, it will automatically report to the emergency services and will inform family members via mobile phone/ GSM module. 

This can help provide quick assistance in case of collisions or injuries, avoiding severe damage. 


Our Product is first of its type. People are using simple wooden or metal sticks to move around which is priced at par with Smart Walker. This is going to be a big leap which is going to revolutionize things in a very big way not only by features but by the financial aspects. We have decided to sell this at a minimal cost.

The companion app is divided into 2 parts.

1. For User

2. For Family Members

About the Companion App
(For User)


The user can check health information like the number of steps, calories burnt, real-time heart rate from the companion app. The user will also have the option to toggle different settings regarding privacy and Smart Walker. We understand that most of the users might be blind and might not be able to read the content so the app has several accessibility features. We have a screen reading feature built in the app for the blind.


About the Companion App
(For User's family members)


Using the companion app, family members can track the live location of the stick’s user. This can help by giving the family an assurance about where the user is in real-time. This will also help them make sure they have reached their destination safely.  Family members can directly navigate to Smart Walker’s location via the shortest path. The family members can also call for the nearest assistance to the user with help of emergency services.
